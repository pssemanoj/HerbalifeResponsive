ΑΜΕΣΗ ΠΡΟΒΟΛΗ ΠΑΡΑΓΓΕΛΙΑΣ
Επιλέξτε, προσθέστε, επεξεργαστείτε την αποστολή ή τον τόπο παραλαβής της παραγγελίας σας.


Είδος
Δείχνει αν η παραγγελία είναι για αποστολή ή παραλαβή.

Ψευδώνυμο
Οι τοποθεσίες που χρησιμοποιείτε συχνά για την αποστολή ή την παραλαβή μπορούν να αποθηκευτούν με ψευδώνυμο για ευκολία στη χρήση. Την επόμενη φορά που θέλετε να χρησιμοποιήσετε την ίδια θέση, απλώς επιλέξτε την από τη λίστα με τα ψευδώνυμα.


ΠΡΟΒΟΛΗ ΠΕΡΙΓΡΑΦΩΝ ΑΠΟΓΡΑΦΗΣ

Εμφάνιση όλων των προϊόντων: Όταν είναι επιλεγμένο, εμφανίζονται όλα τα προϊόντα ανεξάρτητα από τη διαθεσιμότητά τους.
Εμφάνιση όλων των διαθέσιμων προϊόντων: Όταν είναι επιλεγμένο, εμφανίζονται μόνο τα προϊόντα της παραγγελίας που είναι διαθέσιμα για την συγκεκριμμένη αποστολή ή στον τόπο παραλαβής.

Υπόμνημα περιγραφής διαθεσιμότητας
ΠΡΑΣΙΝΟ - Το προϊόν είναι διαθέσιμο για παραγγελία με αποστολή ή παραλαβή.

ΚΙΤΡΙΝΟ- Τα προϊόντα αυτό έχει εξαντληθεί αλλά μπορεί να παραγγελθεί για αποστολή, όταν το προϊόν είναι διαθέσιμο, θα αποσταλεί. Προϊόντα σε αυτήν την κατάσταση δεν μπορούν να προστεθούν σε παραγγελίες για παραλαβή.
ΚΟΚΚΙΝΟ- Το προϊόν δεν είναι διαθέσιμο για παραγγελία με αποστολή ή παραλαβή.
