ΤΡΟΠΟΙ ΠΛΗΡΩΜΗΣ
Παρακαλώ σημειώστε ότι όλες οι παραγγελίες θα πρέπει να πληρώνονται εξ ολοκλήρου ώστε η Herbalife να στείλει και να απελευθερώσει την παραγγελία. Τα παρακάτω είναι οι αποδεκτές μεθόδοι πληρωμής.

Σχετικά με τις Πληρωμές μέσω Πιστωτικών Καρτών

  • Η πιστωτική κάρτα θα πρέπει να είναι στο όνομα του αγοραστή.
  • Ο αριθμός CVV πρέπει να αναγράφεται για την επιλεγμένη κάρτα
  • Μια πιστωτική κάρτα μπορεί να χρησιμοποιηθεί για την πληρωμή της παραγγελίας
