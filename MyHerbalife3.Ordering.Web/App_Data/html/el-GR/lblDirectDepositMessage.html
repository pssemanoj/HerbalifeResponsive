  • Οι διανομείς μπορούν να αγοράσουν προϊόντα από την Herbalife μέσω τραπεζικού εμβάσματος προκειμένουν να εξοφληθεί η παραγγελία τους.
  • Παρακαλώ τακτοποιήστε την πληρωμή της παραγγελίας σας μέσα σε 24 ώρες για να αποφευχθεί η ακύρωση της.
  • Κλείσιμο μήνα - Κάθε συνεργάτης είναι υπεύθυνος να εξασφαλίσει ότι οι τραπεζικές καταθέσεις ή μεταφορές θα έχουν πραγματοποιηθεί έγκαιρα πρίν το κλείσιμο του μήνα για να εγκριθεί η πληρωμή της παραγγελίας του.
  • Σε κάθε επικοινωνία παρακαλώ να μας αναφέρετε τον κωδικό συνεργάτη και τον αριθμό παραγγελίας σας.
  • Οι παραγγελίες που είναι για αποστολή θα δρομολογηθούν αμέσως μετά την πληρωμή τους.
