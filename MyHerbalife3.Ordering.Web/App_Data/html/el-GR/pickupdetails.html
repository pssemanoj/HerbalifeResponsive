Έγκυρο ID απαιτείται για την παραλαβή. Όλες οι παραγγελίες που ζητηθούν θα πρέπει να παραληφθούν μέσα σε δέκα (10) εργάσιμες ημέρες από την Herbalife από την ημερομηνία επεξεργασίας της παραγγελίας.

* Σημαντικό * Οι παραγγελίες που δεν παραλήφθησαν μέσα στην προθεσμία αυτή μπορεί να επιβαρυνθούν με πρόσθετο κόστος αποστολής ή θα είχε σαν αποτέλεσμα την ακύρωσή της.

Παρακαλώ συμπληρώστε το όνομα και το επίθετο του ατόμου που θα παραλάβει την παραγγελία.
