Por favor notar que todas las ordenes deberán ser pagadas en su totalidad para que Herbalife pueda enviar o entregar órdenes. A continuación se indican métodos de pagos aceptables._GF40

Acerca de pagos con Tarjetas de Crédito._GF41

  • La tarjeta de crédito debe estar a nombre del titular del ID._GF42
  • El código CVV debe de ser ingresado para la tarjeta indicada._GF43
  • Sólo se usara una tarjeta de crédito para pagar la orden._GF44
