Le paiement par Bancontact/Mister Cash de votre commande Herbalife est effectué via le fournisseur de service de paiement Envoy Services, dont le nom apparaîtra sur votre relevé bancaire.

Lorsque vous avez effectué votre paiement avec Bancontact / Mister Cash, il est important de pouvoir retourner à notre magasin en ligne.
Un bouton spécial existe à cet effet lors du paiement. Si vous n’arrivez pas à retourner à notre magasin en ligne, il est possible que votre commande ne soit pas introduite dans notre système.
Si vous n’avez pas pu avoir la possibilité de retourner à notre magasin en ligne, veuillez prendre contact avec nous par email via l’adresse suivante: InfoMB@herbalife.com Afin de mieux répondre à vos attentes, veuillez nous fournir les informations suivantes dans votre email: numéro ID Herbalife et code PIN de votre myherbalife.com, le montant que vouz avez payé, le nom de votre banque et le numéro de votre compte banacaire.
