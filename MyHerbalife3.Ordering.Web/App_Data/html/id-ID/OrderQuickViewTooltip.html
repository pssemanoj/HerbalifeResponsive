TAMPILAN ORDER CEPAT
Pilih, tambah, ubah atau hapus lokasi pengiriman atau pengambilan barang dari order Anda.


Tipe
Mengindikasikan order melalui pengiriman atua pengambilan barang

Nama panggilan
Frekuensi lokasi pengiriman dan pengambilan barang akan menyimpan nama panggilan untuk memudahkan penggunaan. Waktu berikutnya Anda ingin menggunakan lokasi yang sama, cukup pilih dari daftar nama panggilan.


DESKRIPSI TAMPILAN INVENTORI

Menampilkan semua inventori: Saat memilih, semua pilihan akan ditampilkan tanpa memperhatikan ketersediaan.
Hanya Menampilkan Ketersediaan Inventori: Saat memilih, hanya pilihan yang tersedia untuk di order dengan lokasi pengiriman dan pemngambilan barang spesifik yang akan ditampilkan.

Deskripsi Petunjuk Inventori
HIJAU - Barang akan tersedia untuk order pengiriman dan pengambilan barang.

KUNING- Barang tidak tersedia untuk pengiriman; saat barang kembali tersedia maka akan segera dikirim. Barang pada kondisi ini tidak dapat ditambahkan di order yang akan diambil langsung.
MERAH- Barang tidak tersedia dan tidak akan ditambahkan ke order pengambilan langsung dan pengiriman.
